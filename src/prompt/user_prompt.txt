평가기준
- 각 텍스트의 논제와 논증을 정확하게 독해(파악, 이해)했는가?
- 수업에서 다룬 텍스트들을 종합적으로 분석, 활용하였는가?
- 문장을 어법에 맞게 제대로 구성되었는가?
- 적절한 어휘를 사용하였는가?
- 단락마다 주제가 있는가? 그 주제를 설명 또는 논증하고 있는가?
- 단락간 논리적 연관성이 있는가?
- 전체 글을 논리적으로 구성, 배치함으로써 논리적 설명력을 갖추었는가?
- 자신의 문제의식과 주제가 명료하게 드러나는가?
- 글의 제목이 내용과 주제를 반영하고 있는가?
- 수업에서 다룬 텍스트 외에 다양한 자료를 활용, 인용하여 자신의 주장을 논증하였는가?
- 개인적 성장과정이 드러나는가?
- 경험적, 논리적으로 독자를 설득하는 글인가?
- 독특하고 창의적인 사고의 탁월성을 보여주는가?
